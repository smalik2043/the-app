## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docker

# run through docker

# docker build

$ docker-compose build the-dev

# To start the docker container

$ docker-compose up the-dev

# remove docker container

$ docker-compose down the-dev
